# @amlplugins/google-cloud-build

Native `@google-cloud/cloudbuild` SDK re-exported under the `@amlplugins` namespace.

> Cloud Build.

## Install

```bash
npm install @amlplugins/google-cloud-build
```

## Use

```ts
import { /* named exports from @google-cloud/cloudbuild */ } from "@amlplugins/google-cloud-build";
// or
import sdk from "@amlplugins/google-cloud-build";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
